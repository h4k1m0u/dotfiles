# Install and configure I3
```sh
$ pacman -S xorg-xinit xorg-server
$ pacman -S i3-wm
$ echo 'exec i3' > ~/.xinit.rc
$ startx
```

# Install terminal emulator
```sh
$ pacman -S terminator
```

# Install status bar
```sh
$ pacman -S i3status
```

# Install dmenu
```sh
$ pacman -S dmenu
```

# Install file manager
```sh
$ pacman -S ranger
```

# Install mouse drivers
```sh
$ pacman -S xf86-input-synaptics
```
